# worldcup-games

GitHub Pages'e hazır statik site dosyaları.

## Yayına alma

1. Bu zip'in içeriğini GitHub'daki `worldcup-games` reposunun kök dizinine yükle.
2. Repo içinde `index.html` en üstte görünmeli.
3. GitHub'da `Settings > Pages` bölümüne gir.
4. Source: `Deploy from a branch`
5. Branch: `main`
6. Folder: `/ (root)`
7. Save.

`https://dekervideos.github.io/worldcup-games/`
